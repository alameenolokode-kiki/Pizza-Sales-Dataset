# Pizza-Sales-Dataset
The dataset used in this project contains records of pizza sales made by a restaurant over a period of time. Each record represents a single pizza sale and includes information such as the pizza name, pizza category, pizza size, date of sale, and the amount of revenue generated.

  ## Data Cleaning and Processing
Before analysis, the data was cleaned and processed to ensure accuracy and consistency. Duplicate records were removed, missing values were checked, and pizza names, categories, and sizes were standardized. Date fields were converted into a monthly format for trend analysis, and revenue fields were verified as numeric values. These steps were necessary to eliminate errors and ensure reliable results throughout the analysis and visualization process.
